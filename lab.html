under the hood

No framework, no build, no backend.

terminal engine

A real shell, in vanilla JS

Typeable commands with Tab-complete and ↑/↓ history, over a small virtual filesystem — ls, cat, ps, ping — all hand-rolled, no library.

3d scene

Three.js + UnrealBloom

An orbitable mini-PC that boots into a holographic screen — OrbitControls, bloom post-processing, and a canvas texture as the terminal display.

architecture

One file per page

Each showcase is a single self-contained HTML file. Dependencies pinned via import map to three@0.161.0 on the CDN. Deploys as plain GitHub Pages.

ci guard

Checks on every PR

A no-dependency script runs node --check on every inline script block and verifies every local link target exists; Prettier keeps the tooling formatted.
